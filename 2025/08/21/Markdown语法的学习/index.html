Markdown语法的学习

Markdown 语法全面介绍

一、标题（Heading）

用于定义文本层级，使用 # 符号表示，层级从 #（一级标题）到 ######（六级标题）。

效果：

一级标题

二级标题

二、文本格式化

  1. 加粗：用 ** 或 __ 包裹文本。

    效果：加粗文字

  2. 斜体：用 * 或 _ 包裹文本。

    效果：斜体文字

  3. 加粗+斜体：组合使用符号。

    效果：加粗斜体

  4. 删除线：用 ~~ 包裹文本。

    效果：删除线文字

  5. 下划线：需借助 HTML 标签 <u>。

    效果：下划线文字

三、列表（List）

  1. 无序列表：使用 -、+ 或 * 开头。

    效果：

    • 项目1
    • 项目2
      • 子项目1
      • 子项目2

  2. 有序列表：使用数字加 . 开头。

    效果：

    1. 步骤1
    2. 步骤2
    3. 步骤3

  3. 任务列表（部分编辑器支持）：用 - [ ]（未完成）或 - [x]（已完成）。

    效果：

    • 待办事项1
    • 已完成事项2

四、链接与图片

  1. 链接：

    • 行内链接：[文本](链接地址 "可选标题")

      效果：百度

    • 引用式链接：[文本][标签]，标签在文末定义。

      效果：[谷歌][google]

  2. 图片：

    • 行内图片：![替代文本](图片地址 "可选标题")

      效果：

    • 引用式图片：类似引用式链接，标签定义图片地址。

五、代码与代码块

  1. 行内代码：用 ` 包裹代码片段。

    效果：print("Hello") 是 Python 代码

  2. 代码块：用 ``` 包裹，可指定语言实现语法高亮。

六、表格（Table）

使用 | 分隔列，- 分隔表头和内容，需保证对齐。

效果：

表头1 表头2
内容1 内容2
内容3 内容4

对齐方式：

  • 左对齐：| 内容 |
  • 右对齐：| 内容 |
  • 居中对齐：| 内容 |

七、引用（Blockquote）

用 > 表示引用，可嵌套多层。

效果：

这是一级引用

这是二级引用

这是三级引用

八、分隔线（Horizontal Rule）

用 ---、*** 或 ___ 表示，需单独占一行。

效果：

九、转义字符

若需显示 Markdown 符号本身，用 \ 转义。

效果：

显示井号

  • 显示星号

十、HTML 混合使用

Markdown 支持直接插入 HTML 标签，实现更多样式（如底色、对齐等）。

效果：

这是 HTML 实现的底色块
居中文本

十一、扩展语法（非标准，部分编辑器支持）

  1. 脚注（如 Typora）：

    效果：这是带脚注的文本[^1]

  2. 表情符号（如 GitHub）：

    效果：:smile: :heart: :warning:

总结

Markdown 以简洁的语法实现文本结构化，标准语法覆盖标题、列表、链接、代码等基础功能，复杂样式可通过 HTML 扩展。不同平台（如 GitHub、Typora、掘金）可能支持额外扩展语法，使用时需参考对应文档。
